# 🚀 SoftForge – Software Development Company Website

SoftForge is a modern **Software Development Company Landing Page** built using **HTML5, CSS3, and a small amount of JavaScript**. The project demonstrates how a professional software agency website can be structured with multiple sections such as services, technologies, portfolio, development process, testimonials, and contact.

This project is ideal for **web design beginners, portfolio projects, and learning responsive layout design**.

🔗 https://your-live-demo-link.com

# 🧩 Features

* Modern Software Company Landing Page
* Clean and Professional UI Design
* Fully Responsive Layout
* Mobile Navigation with Hamburger Menu
* Service Cards
* Technology Showcase
* Portfolio Projects Section
* Development Process Section
* Testimonials Section
* Contact Form Section
* Professional Footer with Newsletter Subscription

# 🖥 Website Sections

## 1️⃣ Hero & Header Section

The hero section introduces the company with:

* Logo
* Navigation menu
* Sign-in button
* Headline and description
* Call-to-action buttons
* Hero illustration

A **hamburger menu** is used for mobile navigation.

---

## 2️⃣ Services Section

This section displays the main services offered by the company:

* Web Development
* App Development
* Custom Software Development

Each service includes:

* Image
* Title
* Description

---

## 3️⃣ Technologies Section

Displays the technologies used by the company such as:

* HTML
* CSS
* JavaScript
* React
* Node.js
* Python
* Cloud
* AWS

Each technology is shown in a card layout.

---

## 4️⃣ Portfolio Section

Shows example projects built by the company.

Projects included:

* Business Website
* E‑Commerce App
* SaaS Dashboard

Each card includes:

* Project image
* Technology used
* "View Project" button

---

## 5️⃣ Development Process Section

This section explains the workflow used to deliver software solutions:

1. Planning
2. Design
3. Development
4. Testing

Each step is represented with an image and description.

---

## 6️⃣ Testimonials Section

Displays feedback from clients.

Each testimonial card includes:

* Quote icon
* Client message
* Client photo
* Client name and role

---

## 7️⃣ Contact Section

Allows users to contact the company through a simple form.

Form fields:

* Name
* Email
* Message

Also includes company contact details:

* Address
* Email
* Phone number

---

## 8️⃣ Footer Section

The footer contains multiple elements:

* Company information
* Social media icons
* Quick links
* Services list
* Contact information
* Newsletter subscription form
* Copyright notice

---

# 📱 Responsive Design

The website is fully responsive and adapts to:

* Desktop screens
* Tablets
* Mobile devices

Media queries are used in CSS to adjust layouts for smaller screens.
